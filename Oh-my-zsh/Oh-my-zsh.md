# 安裝Oh-my-zsh

   1. 安裝套件

      ```
      $ sudo apt-get install zsh
      $ sudo wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh
      ```

   2. 變更系統預設的Shell

      ```
      ＄ sudo chsh -s `which zsh`
      ```

   3. 重開機令設定生效

      ```
      $ sudo shutdown -r 0
      ```
